# Palette Anchors — Non-Default Color Systems

Every palette here is a starting point that avoids the default white/gray/blue + one accent. Each includes hex codes and suggested usage.

---

## Tannery
Warm, earthy, leather-inspired. Good for premium, heritage, or craft brands.

```
Primary:      #8B5E3C   (leather brown)
Secondary:    #D4A574   (caramel)
Accent:       #2D1810   (dark oak)
Light:        #F5EDE3   (cream)
Background:   #FAF6F0   (warm white)
```

**Usage:** Headers in Primary, body in Dark Oak, CTAs in Accent, backgrounds in Light or warm white.

---

## Coastal
Deep navy/teal/sand. Professional but not corporate.

```
Primary:      #1B3A4B   (deep navy)
Secondary:    #2A9D8F   (teal)
Accent:       #E9C46A   (sand)
Light:        #F8F9FA   (white)
Error:        #E76F51   (coral)
```

**Usage:** Primary for nav/headers, Secondary for interactive elements, Accent sparingly for highlights.

---

## Riso
Vibrant, slightly imperfect print colors. Good for creative/editorial.

```
Primary:      #E63946   (riso red)
Secondary:    #457B9D   (riso blue)
Accent:       #F1FAEE   (paper)
Dark:         #264653   (ink dark)
Highlight:    #F4A261   (riso orange)
```

**Usage:** Bold statements in Primary, body in Dark, backgrounds in paper.

## Arte Povera
Raw, material, earthy. Good for artistic, fashion, or cultural.

## How to Use

1. Pick one palette as primary
2. If you need extended colors, borrow accent/light from another palette
3. Never use all 5 colors equally — designate Primary (60%), Secondary (30%), Accent (10%)
4. Test contrast ratios — Accent against Primary must pass WCAG AA
5. Dark mode: shift accents to be more saturated, backgrounds less black

# Silbar Security — Mobile Hero & Trust Section Fixes

## Overview

Fix mobile layout issues: hero video not filling screen, trust section video dock too small.

---

## ISSUE 1: Hero Video Not Filling Screen on Mobile

### Problem
On mobile, the video doesn't fill the full screen height. There's empty dark space at the top. The video should fill the entire viewport on mobile.

### Root Cause
- `.hero-left-panel` on mobile is set to `bottom: 0; height: auto;` — it sits at the bottom as a band
- The video has `object-fit: contain` (from earlier fix) — this fits within bounds, doesn't fill
- The video needs to fill 100vw × 100vh on mobile

### Fix

**CSS Change (`hero.css`):**
```css
/* Mobile — video fills full screen */
@media (max-width: 767px) {
  .morph-video {
    position: fixed;
    top: 0;
    left: 0;
    width: 100vw;
    height: 100vh;
    object-fit: cover;  /* fill screen, not contain */
    z-index: 5;
  }
  
  /* Left panel becomes a bottom band over the video */
  .hero-left-panel {
    position: fixed;
    bottom: 0;
    left: 0;
    right: 0;
    top: auto;
    height: auto;
    z-index: 10;
    background: linear-gradient(105deg, rgba(8, 2, 2, 0.98) 0%, rgba(30, 5, 5, 0.92) 60%, rgba(50, 8, 8, 0.6) 100%);
    backdrop-filter: blur(14px);
    border-top: 1px solid rgba(204, 34, 34, 0.3);
    padding: 1rem 1.5rem 1.5rem;
  }
}
```

## ISSUE 2: Trust Section Video Dock Too Small on Mobile

### Problem
On mobile, the video dock (trust-card-light) is too small. The video appears as a tiny box at the bottom. It needs to be bigger and properly positioned.

### Root Cause
- `.trust-card-light` has `aspect-ratio: 3 / 4` on mobile (≤900px) — portrait ratio makes it tall but narrow
- On very small screens (≤480px), it's `aspect-ratio: 4 / 5` — still small
- The video is scaled to fit this small container

### Fix

**CSS Change (`trust.css`):**
```css
/* Mobile — bigger video dock */
@media (max-width: 767px) {
  .trust-grid {
    grid-template-columns: 1fr;
    gap: 1.5rem;
    padding-bottom: 1rem;
  }
  
  .trust-card-light {
    aspect-ratio: 16 / 9;  /* landscape ratio for video */
    width: 100%;
    max-height: 50vh;  /* don't exceed half viewport */
    border-radius: 16px;
  }
  
  .trust-heading-light {
    font-size: clamp(1.8rem, 8vw, 3rem);
  }
  
  .trust-subcopy {
    font-size: 0.75rem;
    margin-bottom: 1.5rem;
  }
}

/* Very small screens */
@media (max-width: 480px) {
  .trust-card-light {
    aspect-ratio: 16 / 10;
    max-height: 45vh;
  }
}
```
